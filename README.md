# Afterburning Turbojet Performance Analysis

MATLAB-based turbojet engine performance analysis project using atmospheric data and cycle calculations.

## Project Description

This project analyzes turbojet engine performance under different flight conditions. The calculations are performed in MATLAB, and atmospheric properties are taken from an external Excel file.

The project includes numerical calculations, performance plots, and a MATLAB App Designer interface.

## Features

- Turbojet cycle calculations
- Atmospheric data import from Excel
- Performance analysis at different flight conditions
- MATLAB-based plotting
- App Designer interface

## Files

- `turbojet.m` - Main MATLAB calculation script
- `atmosphere.xlsx` - Atmospheric data table
- `HARAM.mlapp` - MATLAB App Designer interface

## Tools

- MATLAB
- MATLAB App Designer
- Excel
